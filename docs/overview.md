---

## Назначение

<!-- TODO: [ВОПРОС РАЗРАБОТЧИКУ] Проверить корректность описания назначения -->

Оператор **itcacmistio** обеспечивает интеграцию сервисной сетки Istio с внешним центром сертификации (ACM — Automated Certificate Management). 

Оператор решает следующие задачи:

- Замена встроенного удостоверяющего центра Istio (Citadel) на корпоративный корневой сертификат
- Автоматическая выдача TLS-сертификатов для сервисов в кластере Kubernetes
- Централизованное управление сертификатами через внешние серверы конфигурации (configServers)

---

## Область применения

<!-- TODO: [ВОПРОС РАЗРАБОТЧИКУ] Уточнить область применения -->

Оператор предназначен для кластеров Kubernetes, в которых:

- Развёрнута сервисная сетка Istio
- Требуется использование корпоративного PKI вместо самоподписанных сертификатов Istio

### Сетевые требования

<!-- TODO: [ВОПРОС РАЗРАБОТЧИКУ] Указать порты и протоколы -->

Оператор требует сетевой доступ к серверам конфигурации (configServers):

- Протокол: HTTPS
- Порт: 443
- URL-формат: `https://<hostname>/api/acmcd`
